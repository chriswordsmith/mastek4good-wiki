# Problem 5

How might frontline staff manage reminders, follow-ups and next actions in one place they can trust.
We had some initial questions;

* Are these tasks assigned by team leads or are they self assigned?
* Are cases managed by multiple people?
* Do they use work and/or personal devices?


The requirements/capabilities that we thought were important were:

* Auto Reminders
    * How often?
    * What method?

* Note tracker
* Calender Intergration
    * Include a link to the action or task

* A dashboard to display upcoming tasks
    * Ability to select into that task
    * Maybe a case management dashboard as well

* A way of managing tasks not yet done
    * RAG rating based on urgency

* ASYNC (to handle offline requirements)
    * SMS
    * Email
    * Whatsapp

* Configuration options around preferred reminder methods 
* Next step suggestions
* Confirmation of task completion and/or next step creation
